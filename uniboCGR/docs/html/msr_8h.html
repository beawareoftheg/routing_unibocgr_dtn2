Detailed Description

This file provides the declaration of the function used to start, call and stop the Moderate Source Routing. You find also the macro to enable MSR in this CGR implementation, and other macros to change MSR behavior.

Boolean: set to 1 (enable) if this node has the complete knowledge of the contact plan, otherwise set to 0. For a "wise node" the MSR must find all the contacts proposed in the MSR route.

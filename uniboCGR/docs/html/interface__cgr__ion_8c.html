Detailed Description

This file provides the implementation of the functions that make this CGR's implementation compatible with ION.

In particular the functions that import the contact plan and the BpPlans. We provide in output the best routes list.

This function is thinked to convert the hops list of a Route and only for this scope.

19/02/20
Returns
int
Return values
>= 0Number of contacts converted
-1CGR's contact NULL
-2Memory allocation error
-3Contact not found in the ION's contacts graph
Parameters
[in]ionwmThe partition of the ION's contacts graph
[in]*ionvdbThe ion's volatile database
[in]CgrHopsThe list of contacts of this CGR's implementation
[out]IonHopsThe list of contact in ION's format.

ionvdb doesn't have to be NULL

  3. To the ION's contact this function adds the citations to the hop of the list where is the contact.
